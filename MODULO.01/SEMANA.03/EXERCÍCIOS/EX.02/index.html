Módulo 01 - Semana 03 - Exercício 02

Em um arquivo app.js (incluído em index.html) faça o seguinte:

No documento js faça uma arrow function para retornar uma mensagem (string) de "Olá, Mundo!".

Armazene esta arrow function em uma constante (const) de nome mensagemOla.

Chame a função mensagem Ola e exiba o retorno desta função no console.

Ajuste a função mensagem Ola para que receba um parâmetro nome e retorne a mensagem "Olá, [nome]!".

Exemplo:

const resultado = mensagem Ola("Mariana"); console.log(resultado); // "Olá, Mariana!"

Ao final, devemos ter uma função que monta uma mensagem de olá customizada com o nome informado por parâmetro.

Dica:https://developer.mozilla.org/pt-BR/docs/Web/JavaScript/Reference/Functions/Arrow_functions
